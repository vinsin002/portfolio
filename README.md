# Vikrant Singh | Data Analyst Portfolio

A React.js portfolio showcasing data analysis, ML, and analytics projects from [GitHub](https://github.com/vinsin002).

## Projects Included

- **Flight Delay Analysis & Prediction** – Predictive analytics for airline operations
- **Credit Card Default Prediction** – ML classification for financial risk
- **Dynamic Pricing for Urban Parking** – Optimization modeling
- **Revenue Leakage in Hospitality** – Business intelligence & profitability
- **Inventory Inefficiencies (SQL)** – Advanced SQL analytics
- **Emotion Classification (Speech)** – NLP & audio ML
- **Summer Analytics Assignment** – EDA & statistics
- **AnnotatorPro** – Data annotation web app

## Run Locally

```bash
npm install
npm run dev
```

Then open [http://localhost:5173](http://localhost:5173).

## Build for Production

```bash
npm run build
```

## Tech Stack

- React 18
- Vite
- CSS (no framework)
